Movie System 🎬
A single-page web application for browsing, reviewing, and managing movies & series.

🚀 Features
Browse a collection of movies & TV shows 📺

User authentication (Login/Register) 🔐

View movie details, including posters and descriptions 🎞️

Responsive design for desktop & mobile 📱

Role-based access control (Admin/User) 👤

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript, Bootstrap

Backend: PHP, FlightPHP

Database: MySQL (with PDO for secure queries)

API: REST architecture with JWT authentication


🛠️ Setup Instructions

Clone the repository

git clone https://github.com/klm66655/web2025.git

Start Backend

cd web2025

cd backend
php -S localhost:8000
